Este proyecto es una aplicación móvil desarrollada en Flutter como parte de la solemne del ramo Desarrollo Android.
Incluye navegación entre pantallas, manejo de assets (imágenes), uso de widgets personalizados, estilos, y apertura de sitios web externos usando url_launcher.
Tecnologías utilizadas
Flutter 3.x
Dart
Material Design
url_launcher para enlaces externos
Manejo de imágenes con carpeta assets/images
